3.启动服务器（由于阿里云内存只有1G，所以这里分配的512M）

然后会出现如下错误：

解决办法： 

vim eula.txt  
eula=true (默认是false)

vim server.properties  
online-mode=false

重新启动服务器（出现如下的log说明启动成功）

4、为了启动方便我们自己编写一下启动的shell文件
在minecraft安装目录下新建一个 start.sh
shell命令如下：

#!/bin/bash
cd /usr/minecraft/
eval  java -Xmx512M -Xms512M -jar minecraft_server.1.7.10.jar nogui &
# Minecraft
